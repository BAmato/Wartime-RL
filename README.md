# WarTime RL

WarTime RL is a custom **OpenAI Gymnasium** environment designed for research into strategic territory control using Deep Q-Networks (DQN). This environment was designed from scratch by the team as an original simulation, taking advantage of the course option to build a custom environment. The key motivation for this choice is the unique presence of stochasticity in combat outcomes: unlike deterministicenvironments, the dice mechanic means the agent must learn to reason under uncertainty, developing risk-tolerant strategies rather than simply memorizing optimal action sequences.
